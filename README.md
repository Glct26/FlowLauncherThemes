# Flow Launcher Themes
A collection of custom-designed themes for Flow Launcher.

## Installation
1. **Download** the `.xaml` file for the theme you want to use.
2. **Move** the file to the themes directory:
   `%APPDATA%\FlowLauncher\Themes`
3. **Restart** Flow Launcher.
4. **Select** the theme: `Settings` -> `Appearance` -> `Theme`.
5. **Sync Layout:** To ensure correct margins and window scaling, click the **pencil icon** (top right of the Settings window), scroll to the bottom, and click **Import Theme Size**.

## Available Themes
* **Spotless Dark** - Mac-inspired dark theme with optimized Gaussian blur and precise alignment.
* **Spotless Dark Clean** - The minimalist version of Spotless Dark. Subtitles are removed for a completely uncluttered look.
