# Flow Launcher Themes
A collection of custom-designed themes for Flow Launcher.

## Installation
1. **Download** the `.xaml` file for the desired theme.
2. **Move** the file to: `%APPDATA%\FlowLauncher\Themes`
3. **Restart** Flow Launcher.
4. **Select** theme: `Settings` -> `Appearance` -> `Theme`.
5. **Sync Layout:** Click the **pencil icon** (top right of Settings), scroll to the bottom, and click **Import Theme Size**.

## Available Themes
* **Spotless Dark** - Mac-inspired dark theme with optimized Gaussian blur and precise alignment.
* **Spotless Dark Clean** - Minimalist version of Spotless Dark without subtitles.
* **Spotless Light** - Glass-like light theme with optimized opacity for native background reflection.
* **Spotless Light Clean** - Minimalist version of Spotless Light without subtitles.
